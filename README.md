# CatchUpSite

A single-page marketing site for **CatchUp — Smarter decisions for tomato growers**, built with [SvelteKit](https://kit.svelte.dev/) and prerendered to fully static HTML, plus the Terraform and GitHub Actions wiring that hosts it on AWS (S3 + CloudFront).

## Project structure

### Folders

#### Source (the actual project)

| Folder | Purpose |
|--------|---------|
| `src/` | SvelteKit application source code. |
| `src/routes/` | Page routes — each subfolder is a page in the site. |
| `src/routes/features/` | The "features" page route. |
| `src/routes/rules/` | The "rules" page route. |
| `src/lib/` | Reusable Svelte components (Logo, Nav, Footer, Countdown). |
| `src/lib/assets/` | Static assets imported into components (e.g. `tomato.webp`). |

#### Infrastructure & CI/CD

| Folder | Purpose |
|--------|---------|
| `terraform/` | Terraform config that provisions the AWS hosting (S3 + CloudFront + OIDC IAM). |
| `terraform/.terraform/` | Terraform's local cache of downloaded providers (the AWS provider) — auto-generated. |
| `.github/` | GitHub repository configuration. |
| `.github/workflows/` | GitHub Actions pipelines — `deploy.yml` (build + deploy on push) and `terraform-plan.yml` (plan check on PRs). |

#### Tooling & generated (not committed source)

| Folder | Purpose |
|--------|---------|
| `.claude/` | Claude Code project settings. |
| `.git/` | Git version-control metadata. |
| `.svelte-kit/` | SvelteKit's generated dev/build internals — types, server entries, client output — regenerated by the build, not edited by hand. |
| `build/` | The final prerendered static site (the deployable artifact synced to S3). |
| `node_modules/` | Installed npm dependencies. |

### Root files

| File | Purpose |
|------|---------|
| `package.json` | Declares the project's npm dependencies and scripts (`dev`, `build`, `preview`). |
| `package-lock.json` | Locks exact dependency versions for reproducible installs (`npm ci`). |
| `svelte.config.js` | SvelteKit configuration — sets up `adapter-static` to prerender the site to static HTML. |
| `vite.config.js` | Vite build/dev-server configuration (the bundler SvelteKit runs on). |
| `jsconfig.json` | JavaScript/editor config — path aliases and type-checking hints for IDE support. |
| `bootstrap.sh` | One-time script that provisions the S3 bucket for Terraform state (already run; don't re-run). |
| `.gitignore` | Lists files/folders Git should ignore (e.g. `node_modules/`, `build/`, `.svelte-kit/`). |
| `CLAUDE.md` | Project guidance for Claude Code — describes the repo, architecture, and common commands. |
| `README.md` | This file. |

## Common commands

```bash
npm install     # install dependencies (first time only)
npm run dev     # hot-reloading dev server at http://localhost:5173
npm run build   # build the static site into build/
npm run preview # preview the production build locally
```
